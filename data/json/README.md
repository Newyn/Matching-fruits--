##json (FR)

Ce dossier contient des fichiers de configurations tels que celui des options, des niveaux et des exploits. <br/>
Ils sont chargés à l'initialisation de l'application via une requête XHR (settings.js).

##json (EN)

This folder contains configuration files such as options, levels and achievements. <br />
They are loaded at the initialization of the application via XHR request (settings.js).
